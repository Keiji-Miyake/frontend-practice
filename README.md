## JavaScript

### 命名規則

* 変数／関数名              -> camelCase記法
* 定数名                   -> アンダースコア記法
* クラス（コンストラクター）名 -> Pascal記法

便宜的な変数は「i」「j」などのようにできるだけ短い名前をつけるのが一般的。
呼び出しの利便性を考慮して、頻繁に呼び出す関数には「$」をつけるケースもある。
